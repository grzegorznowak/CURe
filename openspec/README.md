# CURe OpenSpec

This is the OpenSpec root for the CURe project. Keep initiative and change-workspace planning artifacts here, next to the CURe source tree, so `/openspec-*` skills can run with `/workspaces/cure_workspace/projects/CURe` as the workspace root.

- Initiatives: `initiatives/<slug>/initiative.md`
- Change workspaces: `changes/<story-slug>/`
- Story-change schema/templates: `schemas/story-change/`

Legacy coordination history for the migrated subsequent-review initiative remains under `/workspaces/cure_workspace/agent_coordination/epics/cure-subsequent-pr-review/`.
